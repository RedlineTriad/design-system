# @swisspost/design-system-components

[![npm version](https://badge.fury.io/js/@swisspost%2Fdesign-system-components.svg)](https://badge.fury.io/js/@swisspost%2Fdesign-system-components)

## Installation

<!-- https://stenciljs.com/docs/custom-elements#consuming-custom-elements -->

```bash dark
npm install @swisspost/design-system-components
```

## Usage

Import all the components into your project (for example into your `/src/main.ts` or `/src/index.js` file):

```js dark
import { defineCustomElements } from '@swisspost/design-system-components';

defineCustomElements();
```

Import specific components:

```ts dark
import { defineCustomElement as definePostCollapsible } from '@swisspost/design-system-components/dist/components/post-collapsible';

definePostCollapsible();
```

Use the imported components in your html:

```html dark
<post-collapsible>
  <span slot="header">Titulum</span>

  <p>Contentus momentus vero siteos et accusam iretea et justo.</p>
</post-collapsible>
```
