# @swisspost/design-system-styles

[![npm version](https://badge.fury.io/js/@swisspost%2Fdesign-system-styles.svg)](https://badge.fury.io/js/@swisspost%2Fdesign-system-styles)

## Installation

```bash dark
  npm install @swisspost/design-system-styles
```

## Usage

Import the stylesheet into your project (for example into your `/src/styles.scss` file):

```css dark
@use '@swisspost/design-system-styles/<bundle-name>.<ext>';
```

Available Sass entrypoints:

- **index.scss** (internet facing applications)
- **intranet.scss** (internal applications)
- **basics.scss** (atomic styles for font, buttons, lists, etc.)
- **core.scss** (variables, functions and mixins only)

Available CSS bundles:

- **index.css** (internet facing applications)
- **intranet.css** (internal applications)
- **basics.css** (atomic styles for font, buttons, lists, etc.)

If you want to use the SCSS core for your custom styles, we recommend to import & use it as follows, to prevent conflicts:

```css dark
@use '@swisspost/design-system-styles/core.scss' as post;

body {
  background-color: post.$white;
}
```
